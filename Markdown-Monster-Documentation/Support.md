If you have any issues with Markdown Monster you have a couple ways to let us know and interact about it:

* <a href="https://github.com/rickstrahl/markdownmonster/Issues" target="top">File a GitHub Issue in the Markdown Monster Repository</a>   
The source code for Markdown Monster is available in the open, and the Github repository can be used to report bugs, feature requests and any focused usability questions. The Github Issues provide our official bug tracking mechanism so you can also see what issues are open and what we're working on.
    
    Github issues work best for concise bug reports or focused feature requests. Please try to be as concise as possible, provide repo steps as best as possible, and keep any issue to a single topic.


* <a href="https://support.west-wind.com/thread4nm0m17rc.wwt" target="top">Markdown Monster Support Forum</a>  
For more general questions, or open ended discussion of features, or perhaps questions about creating your own add-ins for Markdown Monster, the support forum provides a more open ended discussion forum. It's best suited to longer discussions or indirectly related issues to Markdown Monster's core feature set.
